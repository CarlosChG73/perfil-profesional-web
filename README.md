# Perfil Profesional Web

Proyecto web desarrollado con HTML5 y CSS3 para presentar un perfil profesional orientado a TI, soporte digital, tecnologías web y evidencias de trabajo en formato web y documental.

## Descripción

Este proyecto funciona como una página de presentación profesional y portafolio personal, construida con una estructura semántica clara, diseño visual más sólido y enfoque responsive. Su propósito es servir como evidencia práctica de maquetación web, organización de contenido profesional y presentación estructurada de proyectos y documentos relacionados con el perfil técnico del autor.

La página fue diseñada para mostrar información profesional relevante, habilidades clave, experiencia transferible, datos de contacto y una sección de evidencias o proyectos, con una interfaz visual más madura y coherente con el resto del portafolio.

## Objetivo del proyecto

El objetivo principal de este sitio es servir como:

- evidencia práctica de trabajo con HTML5 y CSS3
- página principal de presentación profesional en formato web
- base estructural para enlazar proyectos y evidencias del portafolio
- muestra de organización visual, jerarquía de contenido y diseño responsive
- proyecto publicable en GitHub como parte de un portafolio personal

## Alcance de esta versión

Esta versión incluye:

- estructura HTML semántica
- hoja de estilos CSS organizada por bloques
- diseño visual renovado con una línea más profesional y contrastes más marcados
- hero principal con fotografía de perfil
- navegación superior y versión responsive con menú hamburguesa
- navegación interna entre secciones
- corrección visual del desplazamiento por anclas internas
- botón flotante para volver al inicio
- integración de favicon
- tarjetas de proyectos con estado y botón visual de referencia
- organización profesional de carpetas y recursos

## Estructura del sitio

La página está organizada en las siguientes secciones:

- Inicio
- Perfil profesional
- Habilidades clave
- Experiencia transferible
- Evidencias y proyectos
- Contacto
- Footer

## Tecnologías utilizadas

- HTML5
- CSS3

## Características principales

- uso de etiquetas semánticas como `header`, `nav`, `main`, `section`, `article` y `footer`
- navegación interna por anclas
- menú principal adaptable a pantallas pequeñas mediante diseño tipo hamburguesa
- diseño visual con degradados, contraste alto, tarjetas y jerarquía tipográfica
- integración de fotografía profesional en la sección principal
- responsive con media queries para desktop, tablet y móvil
- botón flotante de regreso rápido al inicio
- favicon integrado en el documento
- estructura preparada para agregar enlaces reales a proyectos publicados
- presentación de evidencias tanto web como documentales

## Evidencias incluidas en esta versión

La sección de evidencias y proyectos muestra actualmente tres bloques principales:

### Proyecto 1 | Perfil profesional web
Página desarrollada en HTML y CSS para presentar perfil técnico, habilidades, experiencia transferible y evidencias de trabajo.

**Tecnologías:** HTML5, CSS3  
**Estado:** En actualización

### Proyecto 2 | Comunicado interno institucional simulado
Página desarrollada en HTML y CSS para presentar información académica y operativa de forma clara, con jerarquía visual, navegación interna, diseño responsive e integración de contenido institucional simulado.

**Tecnologías:** HTML5, CSS3  
**Estado:** Terminado

### Proyecto 3 | Guía breve de seguimiento académico y operativo
Documento profesional orientado al registro y seguimiento de incidencias académicas y operativas en una institución educativa simulada, elaborado como evidencia de documentación, organización institucional y comunicación escrita.

**Tecnologías:** Documentación operativa, PDF  
**Estado:** Terminado
